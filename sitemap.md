# Sitemap: Armaan Choudhary Portfolio

This outlines the core navigational structure of the portfolio website, based on the Scrapbook Maximalism design mockups.

## 1. Primary Navigation
* **`/` (Home)**
    * Hero Section (Intro & CTA)
    * Featured Work Preview
    * "The Man Behind the Screen" (Brief About)
    * Skills Overview
    * "Let's Connect" Footer CTA

* **`/work` (Selected Work)**
    * Project Gallery Grid (Polaroid-style thumbnails)
    * Filter/Category tags (optional)

* **`/work/[project-slug]` (Project Detail)**
    * *Example: `/work/zola-sky-system`*
    * Project Hero Image
    * The Vision / Overview
    * Tech Stack (Sticky note style)
    * Process & Chaos (Sketches and workflow)
    * Technical Challenges & Solutions
    * Analog Inspiration / The Lab
    * Next Project Navigation

* **`/about-contact` (About & Contact)**
    * "The Story So Far..." (Detailed Biography)
    * "My Toolbox" (Interactive skills showcase)
    * "Say Hello" (Contact Form)
    * Social Links / Resume Download
